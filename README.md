# 烟台市创新平台申报与评价系统  

## Features

- [x] 市级管理员账号，可查看所有的企业数据，可通过WEB页面控制系统的全部功能
- [x] 区县级管理员账号，可查看各自区县的企业数据，可通过WEB页面控制系统的部分功能
- [x] 专家账号，可根据企业上传的证明文件审核各企业数据，并进行定性打分
- [x] 企业账号，可提交数据并上传证明文件
- [x] 系统自动计算出各企业的定量得分
- [x] 可汇总全部或部分数据，导出为Excel文件
- [ ] 可通过短信或邮箱通知企业审核进度（有时间再弄）

## 1. 项目简介

烟台市创新平台申报与评价系统是一个旨在提升烟台市创新平台申报与评价效率和质量的在线平台。该系统将为创新平台申报和评价过程提供软件支持，以实现更高效、更准确的申报和评价，从而推动烟台市创新平台建设和发展。

该系统将提供以下功能：

1. 创新平台申报功能：支持在线提交创新平台申报材料，包括基本信息、运营模式、产业领域、创新能力等。
2. 创新平台评价功能：支持对已申报的创新平台进行评价，包括创新能力、技术水平、市场前景等方面。
3. 评审人员管理功能：支持评审人员的管理、分配和评审流程的监督和管理，以确保评审的公正性和准确性。
4. 数据分析和报告生成功能：支持数据分析和报告生成，包括申报数量、评审通过率、创新平台分布情况等，以便于管理者对创新平台建设和发展进行分析和决策。

该系统的优势包括：

1. 提高申报和评价效率：通过在线申报和评价功能，可以大大提高申报和评价的效率，减少手工操作和沟通的时间和成本。
2. 提高申报和评价质量：通过规范化的申报和评价流程和评审标准，可以提高申报和评价的准确性和公正性，从而提高申报和评价质量。
3. 提供数据支持：通过数据分析和报告生成功能，可以提供更多的数据支持和决策依据，以便于管理者对创新平台建设和发展进行分析和决策。

该系统将为烟台市创新平台建设和发展提供更加高效、准确和规范的申报和评价支持，有望成为烟台市创新平台管理的重要工具和平台。

## 2.需求分析

1. 市级管理员账号：

- 可以查看全部企业数据，包括各区县的企业数据。
- 可以通过WEB页面控制系统的全部功能，如添加、编辑、删除企业账号、专家账号、区县管理员账号等

1. 区县级管理员账号：

- 可以查看各自区县的企业数据，但无法查看其他区县的企业数据。
- 可以通过WEB页面控制系统的部分功能，如添加、编辑、删除企业账号、专家账号等，但不能添加或删除区县管理员账号。

1. 专家账号：

- 可以根据企业上传的证明文件审核各企业数据，并进行定性打分。
- 可以查看自己负责审核的企业数据，但无法查看其他专家审核的数据。
- 可以对企业数据进行评价和建议，以便企业改进和提高。

1. 企业账号：

- 可以提交数据并上传证明文件，包括企业基本信息、业务范围、业务规模、经营状况等。
- 可以查看自己企业的审核进度和审核结果，以便及时了解自己的企业状况。
- 可以修改自己企业的信息，并重新提交审核。

1. 系统自动计算出各企业的定量得分：

- 系统根据专家的审核结果和评分，自动计算出各企业的定量得分，以便管理员查看和决策。

1. 可汇总全部或部分数据，导出为Excel文件：

- 管理员可以根据需要汇总全部或部分数据，并导出为Excel文件，以便进行分析和决策。

1. 可通过短信或邮箱通知企业审核进度：

- 可以在审核过程中，通过短信或邮箱通知企业审核进度和审核结果，以便企业及时了解自己的审核状态和结果。

## 2. 总体描述

### 2.1 产品功能和特点

该系统旨在提升烟台市创新平台申报与评价的效率和质量，具有以下主要功能和特点：

1. 提供市级、区县级管理员账号、专家账号和企业账号，以实现不同权限的用户登录和使用。
2. 市级管理员账号可查看所有企业数据并控制系统全部功能；区县级管理员账号可查看各自区县的企业数据，但只能控制部分功能；专家账号可审核企业数据并定性打分；企业账号可提交数据并上传证明文件。

### 2.2 用户特点和需求分析

根据不同用户的特点和需求，可分为以下四类用户：

1. 市级管理员：主要关注全市范围内的创新平台申报与评价情况，需要查看和控制全部企业数据和系统功能。
2. 区县级管理员：主要关注本区县范围内的创新平台申报与评价情况，需要查看本区县企业数据并控制部分功能。
3. 专家：主要负责审核企业数据并进行定性打分，需要方便、快捷地了解和审核自己负责的企业数据。
4. 企业：主要提交数据并上传证明文件，需要了解自己的审核进度和审核结果，以便及时了解自己的企业状况

系统界面设计应遵循简洁、清晰、易用、美观的原则，以满足用户的操作需求和审美需求。具体的交互流程应包括以下步骤：
